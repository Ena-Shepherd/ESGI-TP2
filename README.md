# ESGI-TP2
This Series of exercices tends to let you play with loop functions in C. <br/>

>n.b. Le projet est en Francais

## Dependencies
  None ! <br/>
  It is still recommended to use the `Makefile` located in the build folder.

## Compiling with `Makefile`
```sh
cd build
make
./tp2
```
